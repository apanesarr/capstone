## Installation

### 1. Server Code

The server code has no dependencies other than ```python 3```:

Linux
```bash
sudo apt-get install python3 python3-pip    # Ubuntu/Debian
sudo pacman -S python3 python3-pip          # Arch/Manjaro
```

MacOS
```bash
brew install python3
```
